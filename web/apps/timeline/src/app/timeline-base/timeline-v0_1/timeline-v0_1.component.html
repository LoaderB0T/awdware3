@if (lang() === 'de') { Die Prämisse war damals schon so viel wie möglich selbst zu machen und nicht auf fertige Lösungen zurückzugreifen, um möglichst viel zu lernen. The fact that it doesn't make sense to write a forum yourself doesn't matter in this case. }

@if (lang() === 'de') { Die Seite hatte ein Login-System, welches mit PHP und MySQL umgesetzt wurde. Die Passwörter wurden (relativ unsicher) gehashed und gespeichert. The passwords were (relatively insecurely) hashed and stored. There were also some very obvious security holes, which for example led to the fact that you could log in without a password. }

@if (lang() === 'de') { Insgesamt hatte die Seite viele "architektonische" Probleme, welche sich durch die fehlende Erfahrung und das fehlende Wissen ergeben haben. Außerdem war die Seite sehr langsam, da die Datenbankabfragen sehr ineffizient waren und die Seite bei jedem Aufruf komplett neu generiert wurde.
